### 基本常识

#### 模型与 API 相关

- **Model（模型）**。
- **Message（对话历史）**：包含用户输入和先前 AI 的输出。
- **System Prompt（系统提示词）**：给系统的全局指令，可以决定 AI 输出格式。
- **Token（词元）**：相当于 AI 的特殊字符编码，一个 Token 表示一个/多个字符，组合而成就拼成了 AI 的输入和输出。
- **Max Tokens**：最大输出 Token 数量。
- **Context Windows（上下文窗口）**：一般以 Tokens 计。表达模型能够容纳的上文 Tokens 数量。此数值越高不一定越好，还需要检查上下文记忆能力（如 Gemini 全系标配 1M 上下文但记忆效率明显不如 Claude 系列）。
- **Temperature（温度）**：输出的随即程度，越高越发散。
- **Top P（核采样）**：只从累积概率达到 p 的候选 token 中采样。
- **Stream（流式输出）**：是否逐 Token 输出。

#### 功能特性

- **Vision（视觉）**：是否支持图片输入。
- **Function Calling（函数调用/工具使用）**：是否通过自定义指令调用工具。
- **Structured Output（结构化输出）**：按照 JSON 格式输出。
- **Multimodal（多模态）**：支持多种模态输入，如音频、视频等。

### 使用方式

了解主流的使用 AI 的方式。

#### 官方订阅制

##### 国内 AI

大多需要一个能够正常使用的 +86 手机号，有些同样支持邮箱。

如果需要订阅服务，大多微信或 QQ 即可使用。

##### 国外 AI

首先需要一个支持非香港、澳门、台湾节点的 VPN^[1]。推荐使用新加坡或美国西部节点，延迟低且解锁范围广。

如果需要订阅服务，请在网上购买虚拟卡（国内电商平台即可购买，注意查看汇率，选稳定性高、汇率低的。购买前联系客服查看对国外 AI 订阅服务的支持情况）。

需要注意自身 IP 的纯净度，选稳定的 IP，不要乱改节点。

^[1]: 目前国外 AI 对于中国地区的管控非常严格，港澳台地区仍然无法使用。

#### API 调用制

> [!attention] 注意
> 
> API 调用制大多无法使用官方方便的对话服务，需要使用第三方软件调用或自行编写代码调用。但是可带来更多灵活性（使用更多第三方软件的附加功能）。
> 
> 另外，如果使用量较大，API 调用的费用一般比订阅制贵不少。但是对于使用量不稳定的人群，使用 API 可以节省成本。

##### 国内/外 官方 API

一般较贵。国外 API 同样需要 VPN，大多还需要虚拟卡。个人用户不推荐。

##### 中转 API

**这是一个巨大的坑**。*一定要自己辨别之后再下单*。

中转 API 顾名思义，就是通过中转服务器将对服务器的 API 调用中转到官方 API 当中。实则不然，有些是模拟 API 调用。它在官方网站处注册大量的订阅制账号，随机依靠订阅制账号便宜的优势模拟 API 调用给用户用，自然卖的比较便宜。（特别突出的就是 Claude 系列模型，便宜的几乎全是订阅账号使用 Claude Code 并模拟 API 调用）也有白嫖官方 API 免费额度的。

但是，这种方式非常不稳定。官方免费额度政策一旦改变，或是开始大面积封杀此类账号，中转站的服务就会突然不可用，或是高度抬价。

另外，中转站的后端是可以修改模型路由的，很有可能你调用高级模型，钱也是按照高级模型算，但是

因此，一定要选择口碑良好、较稳定的中转站，但是这种一般价格本身就很贵。如 [OpenRouter](https://openrouter.ai/)，本身价格就与上游厂商同步了，但是较为稳定且无需 VPN。还有部分免费模型哦。
